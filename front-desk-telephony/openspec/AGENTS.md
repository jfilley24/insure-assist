# OpenSpec Agent Instructions
When the user mentions "planning," "proposal," or "spec":
1. Open this file to refresh on the protocol.
2. Use the native Antigravity UI to create `task.md` and `implementation_plan.md` artifacts.
3. Do not modify `src/` until the user approves the artifact in the UI.
4. After implementation is verified, generate a `walkthrough.md` artifact in the UI to document completion.
5. Export the final `task.md`, `implementation_plan.md`, and `walkthrough.md` to a new subfolder like `docs/history/YYYY-MM-DD-task-name/` for permanent record.
